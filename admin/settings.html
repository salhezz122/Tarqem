إعدادات المشروع

هذه الصفحة تذكير سريع بالمطلوب لتشغيل Firebase Security Rules ورفع المشروع على GitHub.

1) ضع إعدادات Firebase

افتح الملف: assets/js/firebase.js والصق config الخاص بمشروعك (الموجود في الصورة).

بعد ذلك فعّل Authentication (Email/Password) وفعّل Firestore و Storage.
2) قواعد Firestore / Storage المقترحة

انسخ القواعد من ملف FIREBASE_RULES.md المرفق داخل المشروع.

3) نشر المشروع

يمكنك نشره على GitHub Pages أو Firebase Hosting. الشرح موجود في ملف README.md.
